Innovation Factory

About the client

Innovation Factory was founded in 2005 and has been developing its award-winning idea management software and Challenge approach ever since. It has been recently acquired by KPMG which enabled them to deploy their knowledge and expertise in a broader area.
Innovation Factory is the supplier of PIT® (Power Ideas Together), the innovation software that is used globally by leading companies to support innovation. PIT enables companies to gather, share, enrich and develop ideas, both from inside and outside the company. PIT is used by a broad range of clients such as Heineken, Vodafone, PostNL, Strukton Rail, ProRail, Randstad, Liander, and Heerema Marine Contactors.

Project Description

My main role in producing Innovation Factory's public website was to take the designs mockups provided by their designer and turn them into a fully functional/responsive WordPress site. I converted their designs into a W3C-valid HTML/CSS version using HTML5, CSS3 and jQuery, then implemented it into a Wordpress theme using Genesis Framework (a secure, solid and search-engine-optimized foundation for building Wordpress themes).

Another important aspect of the project was to develop a custom Wordpress plugin to extend Wordpress functionality to allow them easily manage all the site content though the Wordpress admin dashboard, this includes blog articles, slideshow images, their partners, their clients, team members and some custom designed pages with in-page navigation. The plugin was developed using PHP5 according to Wordpress plugin development standards and best practices.

The final phase of the project involved designing and developing a custom newsletter email template (to be used in Mailchimp) that matches the website look and feel.
