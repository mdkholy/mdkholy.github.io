What I do

I offer my services as a front-end/backend developer or web consultant for hire to work with individuals, businesses, agencies, designers and other freelancers for web projects.

Web Development

I will bring your website to life and make it perform. From simple HTML to complex web applications. I use the latest web technologies to ensure your website is future-friendly and supported by all major browsers today.

Enterprise-Level Applications

I build robust, secure, test-driven web applications custom-tailored to your budget, objectives, and value proposition. My preferred toolkit for building new web applications is PHP, AngularJS, HTML5 and CSS3.

WordPress Development

These days, I primarily work with WordPress, the ever-popular Content Management System (CMS) where I build custom and complicated applications very easily with its powerful API. I also provide Wordpress customization, custom theme and plugin development services.

E-commerce Solutions

Online stores number is growing every day. Online shopping is simple and convenient, and starting a business in this field can be much easier and handy. I provide custom e-commerce solutions, most of them based on open-source platforms, like Magento and WooCommerce.

Web services and APIs

I build systems that integrate and offer data via clean APIs. I have implemented many web services and APIs (i.e. SSO Authentication, Twitter and Facebook APIs) into WordPress and other open source applications.

Responsive Web Design

If you own a business, service or web store, mobile compatible website is a must. I can help you with that. This way you’ll be able to offer your iPhone, iPad or Android fans the same quality web experience.

The Glen Echo Cheese Kiosk "Discover A New Cheese" is a responsive web app, custom designed and built to run on iPad and iPod Touch devices.
